# How to submit issues
Submit issues with the following schema in the issue title.

###'issue type' group
type:bug

type:feature

type:idea

type:invalid

type:support

type:task

###'issue priority' group
prio:low

prio:normal

prio:high

###'issue status' group
(These labels describe an issue's state in a defined workflow.)

status:confirmed

status:deferred

status:fix-committed

status:in-progress

status:incomplete

status:rejected

status:resolved

###'issue information' group
info:feedback-needed

info:help-needed

info:progress-25

info:progress-50

info:progress-75

###'version tag' group
ver:1.x

ver:1.1
